# Red Team AI Agent

An AI-powered cybersecurity analysis platform designed for authorized security testing.

## Features
- Reconnaissance guidance
- Vulnerability analysis
- Exploit research (theoretical)
- Report generation
- Defensive strategy recommendations

## Tech Stack
- Next.js
- React
- Tailwind CSS
- Anthropic API
- Vercel deployment

## Legal Notice
This tool is for **authorized security testing only**.
Unauthorized use is illegal.

## Deployment
1. Clone repo
2. `npm install`
3. `npm run dev`
4. Deploy on Vercel
